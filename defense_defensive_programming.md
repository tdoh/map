# 防禦-防禦性程式開發

## 介紹

所有弱點的根源，除了失誤的配置，就是來自於沒寫好的程式。 一個只要求程式能動的工程師是永遠無法成為攻城獅的！<br>
如何在撰寫時避開漏洞，並探查可能的弱點，是成為一個高階開發者的必備技能。
